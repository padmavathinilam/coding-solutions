## Problem

### Create Your First HTML Framework

 **Task:**  Build a Minimal HTML Structure Using `<html>` and `<head>`

 **Instructions:**  You're now ready to put your knowledge of the `<html>` and `<head>` tags into practice by creating a clean, minimal HTML framework.

- Start your document with the correct <!DOCTYPE> declaration to tell the browser you're using HTML5.
- Create the root <html> element.
- Inside the <html> element, add a <head> section.
- In the <head> section, include: A <title> for your page (e.g., "My First HTML Page").

 **What you should see:** 

Click  **Run**  to preview on the right, then hit  **Submit**  when you're ready.
