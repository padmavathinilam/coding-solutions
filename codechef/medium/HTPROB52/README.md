## Problem

### Task - Add Line Breaks to Conference Badge

 **Goal:**  The HTML for the conference badge is already written. Your task is to add `<br>` tags to make the badge display exactly like the below.

 **Requirements:** 

- The full HTML code for the badge is provided.
- Observe the badge image carefully and figure out where line breaks are needed.
- Add <br> tags in the appropriate places.

## Solution

**Language:** C++  
**Runtime:** N/A  
**Memory:** N/A  
**Submitted:** 2026-07-06T11:01:23.243Z  

```cpp
<!DOCTYPE html>
<html>
<head>
  <title>Conference Badge</title>
</head>
<body>
  <h1>Tech Summit 2025</h1>
  <!-- Add one break statement in this p tag -->
  <p>
    Attendee: ALEX JOHNSON<br>
    Company: INNOVATE CORP
  </p>
  
  <h2>Special Requirements</h2>
  <!-- Add one more break statement in this p tag -->
  <p>
```
