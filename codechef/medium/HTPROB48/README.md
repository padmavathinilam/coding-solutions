## Problem

### Task Practice Paragraph Tags

 **Goal:**  Use `<p>` tags to create properly spaced paragraphs in HTML.

 **Instructions:**  In the editor:

- Use <!DOCTYPE html>
- Add <html>, <head> with <title>My Paragraph Page</title>
- Inside <body> add:
- An <h1> with: My Favorite Activity
- A <p> with: 2-3 sentences describing why you like it
- An <h2> with: How to Get Started
- A <p> with: 2-3 steps
