## Problem

### Practice - Structure Content with Headings

 **Task:**  Use the correct heading tags to structure a short, fictional blog post about your favorite movie.

 **What to do:** 

- Use an <h1> tag to create the main title with the text: My Favorite Movie
- Use an <h2> tag for a major section with the text: Synopsis
- Use an <h3> tag for a subsection with the text: The Hero's Journey

 **Example Output:**

## Solution

**Language:** C++  
**Runtime:** N/A  
**Memory:** N/A  
**Submitted:** 2026-07-06T10:51:17.963Z  

```cpp
<!DOCTYPE html>
<html>
  <head>
    <title>My Movie Review</title>
  </head>
  <body>
    <h1> My Favourite Movie</h1>
   <h2> Synopsis</h2>
  </body>
   <h3>The Hero's</h3>
</html>
```
