## Problem

### Structure the Page Body

 **Goal** : Learn how to use the `<body>` tag to place visible content on a webpage.

 **Instructions:** 

- Set the Title: Inside the <head> section, you must use the opening <title> and closing </title> tags. Place the following text between them: My First Web Page
- Set the Body Content: Inside the <body> tags, write the text: I have written html code from scratch!

 **Output Example:** 
The webpage should display:

## Solution

**Language:** C++  
**Runtime:** N/A

```cpp
<!DOCTYPE html>
<html>
  <head>
   <title>
  </head>
  <body>
    <p> I have written html code from scratch!
  </body>
      My First Web Page
   </title>
</html>

```
